# Smart Promo Hub

منصة Smart Promo Hub هي أول منصة عربية متكاملة لإدارة الحملات الإعلانية على Pi Network. 
تم تصميم المنصة لتسهيل إدارة الحملات عبر 10 منصات تواصل اجتماعي، مع دعم لـ 5 لغات ونظام ذكاء اصطناعي متطور لتوليد النصوص الإعلانية.

## مميزات التطبيق
- **إدارة مركزية:** تحكم في جميع حملاتك من مكان واحد.
- **تكامل مع Pi Network:** ربط مباشر وآمن مع المحفظة.
- **ذكاء اصطناعي:** توليد نصوص إعلانية احترافية بضغطة زر.
- **تعدد اللغات:** دعم كامل للغة العربية واللغات الأخرى.
